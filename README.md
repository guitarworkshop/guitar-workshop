# 吉他工坊 React V2

這是 React 正式版 V2。

## 部署
1. 用 GitHub Desktop 覆蓋專案資料夾。
2. Commit。
3. Push origin。
4. GitHub → Settings → Pages → Source 選 GitHub Actions。
5. GitHub → Actions 等綠勾。

如果 `.github` 沒有被上傳：
到 GitHub 網頁 Add file → Create new file，檔名輸入 `.github/workflows/deploy.yml`，內容複製 `DEPLOY_YML_COPY.txt`。
